PokeDex

Projeto para recriar uma Pokédex, utilizando a PokeAPI para trazer informações detalhadas sobre os Pokémon escolhidos. Os usuários terão a capacidade de pesquisar Pokémon tanto pelo número quanto pelo nome na Pokédex.

Estou utilizando um conjunto de tecnologias, incluindo Django para a estrutura do back-end e Python como linguagem principal. Estou trabalhando na implementação do front-end usando HTML, CSS e JavaScript para oferecer uma experiência interativa e amigável aos usuários.

Project Pag

Projeto de landing page que oferece um formulário para integração com a API de CEP. Ao inserir o CEP, os campos são automaticamente preenchidos com as informações correspondentes obtidas pela API. Além disso, implementei uma validação para identificar a bandeira do cartão ao inserir o número, exibindo a bandeira correspondente. Esse projeto foi desenvolvido utilizando Laravel e Vue.js.
